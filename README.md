# Описание проекта
Тема: Применение алгоритмов машинного обучения для анализа валютной пары KZT/USD

Целью данного проекта является разработка модели прогнозирования курса валютной пары KZT/USD с использованием алгоритмов машинного обучения на основе исторических данных и анализа новостного фона. В отличие от стандартных подходов, ограничивающихся временными рядами, в проекте предполагается интеграция текстовых данных из новостных источников, что позволит учитывать влияние экономических и политических событий на валютные колебания.

Источниками данных выступают:

* исторические котировки валютной пары KZT/USD, доступные на [Investing.com](https://ru.investing.com/currencies/usd-kzt-historical-data)

* новостные публикации с сайта [Kapital.kz](https://kapital.kz/news)

Собранные данные проходят этап парсинга и предобработки: нормализация временных рядов, лемматизация текстов, выделение ключевых тональностей новостей (sentiment analysis). Далее формируется интегрированный датафрейм, в котором каждому дню сопоставляется курс, прогноз и соответствующий эмоциональный фон новостного потока.

На основе этих данных обучаются и сравниваются различные модели машинного обучения — линейная регрессия, XGBoost, LSTM — для выявления наилучшего предсказателя краткосрочного изменения курса.

Новизна проекта заключается в комбинации числовых и текстовых факторов для прогнозирования валютного курса, что позволяет учитывать не только статистические закономерности, но и поведенческие реакции рынка на информационный фон.

Результаты прогноза могут быть использованы для выдвижения гипотез о целесообразности увеличения или сокращения внешнеторговых операций (импорта и экспорта). Например, при прогнозе укрепления тенге можно рекомендовать активизацию импорта, а при ожидании ослабления — увеличение экспортных поставок. Таким образом, проект сочетает академический интерес с прикладной экономической пользой.
